# 音声ファイル配置フォルダ

このフォルダに以下の音声ファイル（WAV/MP3）を配置してください:

アプリは同名の `.mp3` → `.wav` の順で自動的に探します（例: `alert-low.mp3` があればそれを使用、無ければ `alert-low.wav` を試行）。

## 必要なファイル

- `alert-low.(mp3|wav)` - 低レベルアラート音
- `alert-medium.(mp3|wav)` - 中レベルアラート音
- `alert-high.(mp3|wav)` - 高レベルアラート音
- `alert-critical.(mp3|wav)` - 緊急アラート音
- `success.(mp3|wav)` - 成功音
- `error.(mp3|wav)` - エラー音
- `info.(mp3|wav)` - 情報音
- `system-start.(mp3|wav)` - システム起動音

詳細は [音声ファイル配置ガイド](../../docs/SOUND_FILES_SETUP.md) を参照してください。

## 音声ファイルの準備

音声ファイルを提供いただくか、以下のサイトから無料でダウンロードできます:

- 効果音ラボ: https://soundeffect-lab.info/
- くらげ工匠: https://www.kurage-kosho.info/
- On-Jin: https://on-jin.com/
