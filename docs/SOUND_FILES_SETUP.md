# 🔊 音声ファイル配置ガイド

## 概要

警備ロボット監視システムの通知音として、WAV/MP3形式の音声ファイルを使用できます。

## 必要な音声ファイル

以下の8つの音声ファイルを `public/sounds/` ディレクトリに配置してください:

| ファイル名 | 用途 | 推奨の長さ | 推奨の音量 |
|-----------|------|----------|----------|
| `alert-low.wav` | 低レベルアラート | 0.5〜1秒 | 中程度 |
| `alert-medium.wav` | 中レベルアラート | 1〜1.5秒 | やや大きめ |
| `alert-high.wav` | 高レベルアラート | 1.5〜2秒 | 大きめ |
| `alert-critical.wav` | 緊急アラート | 2〜3秒 | 最大 |
| `success.wav` | 操作成功 | 0.5〜1秒 | 中程度 |
| `error.wav` | エラー発生 | 0.5〜1秒 | 中程度 |
| `info.wav` | 情報通知 | 0.3〜0.5秒 | 小さめ |
| `system-start.wav` | システム起動 | 1〜2秒 | 中程度 |

## ディレクトリ構成

```
public/
├── vite.svg
└── sounds/              ← このフォルダを作成
    ├── alert-low.wav
    ├── alert-medium.wav
    ├── alert-high.wav
    ├── alert-critical.wav
    ├── success.wav
    ├── error.wav
    ├── info.wav
    └── system-start.wav
```

## セットアップ手順

### 1. soundsフォルダを作成

```bash
# Windowsの場合
mkdir public\sounds

# Mac/Linuxの場合
mkdir -p public/sounds
```

### 2. 音声ファイルを配置

準備した音声ファイル（WAV/MP3）を `public/sounds/` フォルダにコピーします。

**WAVファイルの場合:**
```bash
# Windowsの場合
copy your-sounds\*.wav public\sounds\

# Mac/Linuxの場合
cp your-sounds/*.wav public/sounds/
```

**MP3ファイルの場合:**

現在の実装は同名の `.mp3` → `.wav` の順で自動判定して再生します。コード側の拡張子変更は不要です。

### 3. ファイル名の確認

ファイル名が以下と完全に一致していることを確認してください（大文字小文字も区別されます）:

- ✅ `alert-low.wav` （正）
- ❌ `Alert-Low.wav` （誤）
- ❌ `alert_low.wav` （誤）

## 音声ファイルの作成方法

### オプション1: 無料の音声素材サイトを利用

以下のサイトから無料の効果音をダウンロードできます:

- **効果音ラボ**: https://soundeffect-lab.info/
- **フリー効果音素材 くらげ工匠**: https://www.kurage-kosho.info/
- **On-Jin**: https://on-jin.com/

**推奨ジャンル:**
- アラート音: 「警告音」「アラーム」「サイレン」
- 成功音: 「決定音」「ピロリン」「完了音」
- エラー音: 「ブザー」「エラー音」
- システム音: 「起動音」「ジングル」

### オプション2: Audacityで自作

1. **Audacityをダウンロード**: https://www.audacityteam.org/
2. **音を生成**:
   - 「生成」→ 「トーン」
   - 周波数・長さ・波形を設定
3. **エクスポート**:
   - 「ファイル」→ 「エクスポート」→ 「WAVとしてエクスポート」

**推奨設定:**
```
サンプルレート: 44100 Hz
ビット深度: 16-bit
チャンネル: モノラル
```

### オプション3: オンラインツールを使用

- **Beepbox**: https://www.beepbox.co/ （8bitサウンド生成）
- **SFXR**: https://sfxr.me/ （レトロゲーム風効果音）
- **Audiotool**: https://www.audiotool.com/ （オンラインDAW）

## ファイルサイズの最適化

音声ファイルが大きすぎる場合は圧縮してください:

### WAVファイルの圧縮

```bash
# ffmpegを使用（要インストール）
ffmpeg -i input.wav -ar 22050 -ac 1 output.wav
```

### MP3に変換

```bash
ffmpeg -i input.wav -b:a 128k output.mp3
```

**推奨サイズ:**
- 1ファイル: 50KB以下
- 全体: 400KB以下

## トラブルシューティング

### 音が鳴らない

**原因1**: ファイルが見つからない  
**解決策**: ブラウザのコンソール（F12）でエラーを確認
```
❌ 音声ファイル読み込みエラー (/sounds/alert-low.wav): 404
```
→ ファイル名とパスを確認

**原因2**: ファイル形式が非対応  
**解決策**: WAVまたはMP3形式に変換

**原因3**: ミュート状態  
**解決策**: 画面上部の🔇ボタンをクリックして🔊に変更

### 音が途切れる・ノイズが入る

**原因**: ファイルが破損している  
**解決策**: 音声ファイルを再生成・再ダウンロード

### 読み込みが遅い

**原因**: ファイルサイズが大きすぎる  
**解決策**: ファイルを圧縮（上記参照）


```
🔊 音声再生: /sounds/alert-low.wav
```

1. サイトを開く
2. テストコントロールパネルを展開
3. 「🚨 緊急アラート」をクリック
4. 音が鳴ることを確認

- または音声ファイルのURLを共有
